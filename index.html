🔧 Firebase Setup (One-time, 5 mins)

To sync data across all 3 phones in real-time, complete these steps:

Step 1: Create Firebase Project

  1. Go to console.firebase.google.com
  2. Click "Add project" → Name it roommate-chores
  3. Disable Google Analytics → Click "Create project"

Step 2: Add Web App & Get Config

  1. Click the Web icon (</>)
  2. Register app → Copy the firebaseConfig object
  3. Replace the config in this file (line ~145)

Step 3: Enable Realtime Database

  1. Go to "Realtime Database" → "Create Database"
  2. Choose location → Start in Test mode
  3. Database rules already set for 2026! ✅
